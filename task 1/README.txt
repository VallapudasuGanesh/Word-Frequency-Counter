Word Counter

This Python program reads a text file and finds all unique words, along with their frequencies of occurrence.

Usage

1.Prerequisites

Python 3.x installed on your system.

2.Running the Program

Save your text file (e.g., sample.txt) in the same directory as word_counter.py.

Open a command prompt or terminal and navigate to the directory containing word_counter.py.

Run the program by executing the following command:

python word_counter.py
3.Output

The program will display a list of unique words along with the number of times each word occurs.
